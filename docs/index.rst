Installation
-----------

To install the GoQuant Trading Simulator, follow these steps:

1. Clone the repository
2. Create a virtual environment
3. Install dependencies
4. Configure the application

For detailed installation instructions, see :doc:`installation`.

Usage
-----

The trading simulator provides a user interface for:

- Real-time orderbook data processing
- Market impact analysis
- Slippage estimation
- Maker/Taker proportion prediction

For detailed usage instructions, see :doc:`usage`.

API Reference
------------

The API documentation provides detailed information about:

- WebSocket client
- Orderbook processor
- Trading models
- Performance analysis

For detailed API documentation, see :doc:`api`.

Models
------

The trading simulator implements several models:

- Almgren-Chriss market impact model
- Slippage estimation model
- Maker/Taker proportion prediction

For detailed model documentation, see :doc:`models`.

Performance Analysis
------------------

The performance analysis system provides:

- Processing time tracking
- UI update latency monitoring
- WebSocket latency measurement
- Error rate tracking

For detailed performance analysis documentation, see :doc:`performance`.
